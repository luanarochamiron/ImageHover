# Image Hover Animation with Z-Index Transition

Uma galeria de imagens interativa, moderna e responsiva desenvolvida com HTML5 e CSS3. O projeto apresenta um sistema de exibição de perfis em camadas sobrepostas, onde passar o mouse (hover) sobre os avatares inferiores aciona uma animação fluida de expansão e alteração de prioridade visual da foto correspondente.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica dos contêineres de imagens grandes e dos botões de controle em formato de avatar.
* CSS3: Estilização completa, uso de bordas arredondadas, sombras e gerenciamento de transições.
* Animações Avançadas de CSS: Manipulação dinâmica de `z-index`, `opacity`, `transform` e propriedades de transição para criar o efeito de empilhamento.

## Funcionalidades e Técnicas Aplicadas

* Exibição em Camadas Sobrepostas: Painel central onde as imagens grandes ficam dispostas lado a lado com bordas arredondadas e contornos brancos bem definidos.
* Seleção por Avatares Circulares: Um menu inferior composto por pequenos avatares redondos que funcionam como gatilhos visuais para cada perfil.
* Transição de Z-Index Dinâmica: Uso de animações em CSS puro para trazer a foto selecionada para o primeiro plano, ajustando seu nível de empilhamento vertical sobre as demais.
* Centralização Fluida: Uso de Flexbox para manter todo o componente perfeitamente alinhado no centro da tela sobre um plano de fundo azul sólido de alto contraste.
